Writing Testable Code

What does testable PHP code really look like? The Grumpy Programmer's Guide To Building Testable PHP Applications is a guide to writing PHP code that can be tested using automated testing tools.

Learn about some strategies for refactoring your PHP code in such a way that writing tests becomes easy enough that you quickly make it part of your normal work flow.
